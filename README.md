# ReversiMail – AI Email Assistant with The Reversibility Spectrum

An AI executive assistant that reads, drafts, and sends emails – with step‑up authentication gated by reversibility, not arbitrary categories.

## The Insight: The Reversibility Spectrum
- **Tier 1 – Observe** (read emails) → fully reversible → automatic
- **Tier 2 – Draft** (compose replies) → reversible until finalised → one‑click confirm
- **Tier 3 – Commit** (send emails) → permanently irreversible → step‑up MFA required

## Tech Stack
- Auth0 (Authentication, Token Vault, MFA)
- Next.js 14
- Gmail API
- Tailwind CSS

## Setup
1. Copy `.env.local` with your credentials
2. `npm install`
3. `npm run dev`
